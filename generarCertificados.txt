INSTRUCCIONES PARA GENERAR CERTIFICADOS HTTPS CON MKCERT (Windows + Node.js y Express)

Este archivo explica cómo generar los certificados necesarios (key.pem y cert.pem) para ejecutar el proyecto en modo HTTPS usando mkcert.

--------------------------------------------------
1 Instalar mkcert

1. Descargar mkcert para Windows desde:
   https://github.com/FiloSottile/mkcert/releases
   - Descarga el archivo .exe adecuado para tu sistema (Windows 64-bit).
   - Coloca mkcert.exe en una carpeta accesible, por ejemplo C:\Herramientas\mkcert.
   - Opcional: agrega la carpeta a tu PATH para usar mkcert desde cualquier terminal.

--------------------------------------------------
2 Instalar Root CA local

Abrir PowerShell como administrador y ejecutar:
   mkcert -install

- Esto crea e instala un Root CA confiable en tu sistema y navegador.
- Los certificados locales generados serán confiables para Chrome, Firefox y Edge.

--------------------------------------------------
3  Moverse a la carpeta del proyecto

cd C:\ruta\del\proyecto

Luego entrar a la carpeta donde quieres guardar los certificados:
mkdir cert
cd cert

--------------------------------------------------
4 Generar clave privada y certificado local

Ejecutar:
   mkcert localhost 127.0.0.1 ::1

- Esto genera dos archivos en la carpeta cert/:
   localhost+2.pem       ← Certificado local confiable
   localhost+2-key.pem   ← Clave privada asociada
- Opcional: renombrar a cert.pem y key.pem si quieres mantener consistencia con tu proyecto.

--------------------------------------------------
5 No subir certificados a GitHub

- Los certificados son privados y específicos de tu máquina.
- Asegúrate de que tu .gitignore contenga:
   cert/
   *.pem


--------------------------------------------------
6  Notas importantes

- Los certificados mkcert son solo para desarrollo local.
- Chrome y otros navegadores los reconocerán como confiables si instalaste correctamente el Root CA.
- Nunca compartas tu key.pem fuera de tu equipo.
- Si cambias el nombre de dominio local (ej. de localhost a miapp.local), genera nuevos certificados:
   mkcert miapp.local
